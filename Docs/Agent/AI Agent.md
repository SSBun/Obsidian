# AI Agent 概述

AI Agent（人工智能代理）是能够自主感知环境、进行推理并采取行动以实现目标的自主人工智能系统。它们可以是简单的（反应式）或复杂的（具有记忆、规划和学习能力）。

## 关键特性
- **自主性**：无需持续人工监督即可独立运行
- **目标导向**：朝着特定目标工作
- **适应性**：根据经验学习并调整行为
- **互动性**：与人类和其他代理进行沟通和协作

## AI Agent 类型
- **简单反射代理**：仅对当前感知做出反应
- **基于模型的代理**：维护内部状态表示
- **基于目标的代理**：考虑未来后果
- **基于效用的代理**：最大化预期效用
- **学习代理**：随着时间推移改进性能
